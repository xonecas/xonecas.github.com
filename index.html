function! ToggleBg()
    if &background == "light"
        set background=dark
    else
        set background=light
    endif
endfunction
command! ToggleBg :call ToggleBg()
map <C-X> :ToggleBg<CR>
map <C-C> :NERDTreeToggle<CR>

Sean's vim colorschemes

I hacked these colorschemes together, and will continue to do more. I'm lacking dark and light background colorschmes. I change my background so often that I even have a key binding for it. So this is my effort to get more colors out there.

Just in case you want to get crazy switching backgrounds, add this to your .vimrc. 

function! ToggleBg()
if &background == "light"
set background=dark
else
set background=light
endif
endfunction
command! ToggleBg :call ToggleBg()
map <C-X> :ToggleBg<CR>

These colorschemes were made by fiddling with the source code of Hemisu. They only support GUI vims, (update: cli coming soon!)! The names are random, courtesy of github's repo name sugestions.
